Chapter 4: Sequence Types

- Lists: mutable (can add/remove elements), can contain elements of different types
- Tuples: immutable (cannot be changed after creation), can contain elements of different types
- Strings: immutable (cannot change a character by indexing and assignment or append a char), sequence of characters
- f-strings: formatted string literals, allow embedding expressions inside string literals using curly braces {}

Example built-in string functions:
.split(), .join(), .strip(), .startswith(), .endswith(), .replace(), .upper(), .lower(), .capitalize(), .title()
